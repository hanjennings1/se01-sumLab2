PROJECTS

CALCULATOR LAB

This is a lab assignment for SE01 (Module 8) which is the Summative Lab. This lab is a command-line calculator built with JavaScript that performs basic arithmetic operations using separate functions. It also stores each calculation in an array of objects and allows users to view a history of previous calculations, demonstrating the use of functions, arrays, objects, loops, and basic error handling.

TRIVIA GAME LAB

This project is a command-line (CLI) trivia game built with JavaScript and Node.js. The application presents users with a series of multiple-choice music trivia questions, provides immediate feedback after each answer, tracks the user's score, and records the total time taken to complete the quiz. Throughout the project, I implemented core JavaScript concepts including functions, arrays, loops, array iteration with .map(), asynchronous programming, modules, and user input using @inquirer/prompts.
